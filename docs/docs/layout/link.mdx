# Link

Accessible link with external link handling.

## Usage

```html
<w-link href="https://example.com" external>
  Visit Example
</w-link>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `href` | `string` | `""` | Link destination |
| `external` | `boolean` | `false` | Opens in new tab with security |

## External Links

When `external` is set:
- Opens in new tab (`target="_blank"`)
- Adds `rel="noopener noreferrer"` for security
- Adds visual indicator (optional via CSS)

- Security attributes prevent tabnabbing attacks
