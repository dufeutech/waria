# Context Menu

Right-click context menu.

## Usage

```html
<w-context-menu>
  <w-slot trigger>
    <div>Right-click in this area</div>
  </w-slot>
  <w-slot body>
    <div>
      <w-slot item name="cut"><button>Cut</button></w-slot>
      <w-slot item name="copy"><button>Copy</button></w-slot>
      <w-slot item name="paste"><button>Paste</button></w-slot>
    </div>
  </w-slot>
</w-context-menu>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `open` | `boolean` | `false` | Whether menu is visible |

## Slots

| Slot | Element | Description |
|------|---------|-------------|
| `trigger` | Any | Area that responds to right-click |
| `content` | Container | Menu content |
| `item` | Button | Menu items |

## Events

| Event | Detail | Description |
|-------|--------|-------------|
| `select` | `{ item }` | Item selected |

## Keyboard

| Key | Action |
|-----|--------|
| `ArrowDown` | Next item |
| `ArrowUp` | Previous item |
| `Enter` | Select item |
| `Escape` | Close menu |

## Accessibility

- Same accessibility features as regular Menu
- Opens at cursor position
- Full keyboard navigation
